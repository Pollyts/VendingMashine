Описание решения задания на разработку Web приложения «Автомат по продаже напитков».

Было создано решение asp.net Core с использованием библиотеки React.js. 
С помощью database-first подхода созданы 2 таблицы (папка "Models"), содержащие данные по напиткам и по монетам, которые хранит в себе автомат.
Сам файл базы данных расположен в папке "_Database"
Для обоих классов созданы API контроллеры (папка "Controllers"), принимающие GET, POST, PUT и DELETE запросы с клиентской части приложения и изменяющие данные в базе данных.
В папке "ClientApp" реализована клиентская часть приложения.

Для того, чтобы произошло взаимодействие с бд:
1. Подключиться к базе данных, которая расположена в проекте в папке "_Database"
2. Переименовать базу данных на "vmdb".

Реализовано для пользователя:
1. Возможность внести монеты номиналом 1,2,5,10. Отображение количества внесенных монет
2. Возможность выбора напитков. Отображение стоимости напитка и его изображения. 
3. Возможность приобрести несколько напитков. Ограничение на максимальное количество выбранных напитков (сколько есть напитков в наличии). 
4. Покупка напитков
5. Выдача сдачи. Если монет в автомате недостаточно, будет выдана сдача, максимально приближенная к требуемой (без превышения суммы сдачи).

Реализовано для администратора:
1. Вход по ключу /admin123 
2. Возможность для каждого номинала изменить количество его монет, которое находится в автомате.
3. Возможность заблокировать ввод пользователем любого номинала
4. Возможность добавить новый напиток. Обязательно указывается название, стоимость, количество и изображение.
5. Возможность выбрать напиток и изменить его название, стоимость, количество, изображение
6. Возможность выбрать напиток удалить его.
